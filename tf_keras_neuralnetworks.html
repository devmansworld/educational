Q1: What is TensorFlow?
- TensorFlow is an open-source machine learning framework developed by Google. - Used for building, training, and deploying machine learning and deep learning models.
Q2: Why use Keras?
- Keras is a high-level API for building neural networks easily. - Advantages: simplicity, modularity, wide adoption, and seamless integration with TensorFlow.
Q3: What is the difference between ML and Deep Learning?
- ML: Algorithms learn patterns from data (e.g., linear regression, decision trees). - Deep Learning: Subset of ML using neural networks with multiple layers for complex tasks (e.g., image recognition, NLP).
Q4: What is a neural network?
- Neural networks are computational models inspired by the human brain. - Consist of layers of neurons connected with weights, using activation functions to learn patterns from data.
Q5: How do you install TensorFlow?
- Use pip: pip install tensorflow - Upgrade pip first: pip install --upgrade pip

ðŸ’» TensorFlow & Keras Example

This example classifies handwritten digits using the MNIST dataset.
Python Neural Network Example 
import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

# Load and preprocess MNIST dataset
(train_images, train_labels), (test_images, test_labels) = keras.datasets.mnist.load_data()
train_images, test_images = train_images / 255.0, test_images / 255.0

# Build neural network
model = keras.Sequential([
    layers.Flatten(input_shape=(28, 28)),
    layers.Dense(128, activation='relu'),
    layers.Dropout(0.2),
    layers.Dense(10, activation='softmax')
])

# Compile model
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Train model
model.fit(train_images, train_labels, epochs=5)

# Evaluate model
test_loss, test_acc = model.evaluate(test_images, test_labels)
print(f'Test accuracy: {test_acc}')
Step 1: Preprocessing
- Normalize input images to [0, 1] for faster convergence. - Flatten 28x28 images into 784-dimensional vectors for the dense network.
Step 2: Model Architecture
- Flatten layer: input images reshaped. - Dense layer: 128 neurons, ReLU activation. - Dropout layer: prevents overfitting (randomly disables 20% of neurons). - Output layer: 10 neurons with softmax for class probabilities.
Step 3: Training & Evaluation
- Optimizer: Adam - Loss: sparse categorical cross-entropy - Metrics: accuracy - Epochs: 5 (adjust as needed for better performance)

ðŸ”‘ Key Concepts Summary

TensorFlow Concepts
- Framework for ML and DL. - Supports CPU and GPU acceleration. - Provides low-level and high-level APIs for model building.
Keras Concepts
- High-level neural network API. - Modular, easy to use, integrates with TensorFlow. - Allows building, training, and evaluating models quickly.
Neural Networks Concepts
- Layers: Input, Hidden, Output - Activation functions: ReLU, Softmax, Sigmoid - Loss functions: MSE, Cross-Entropy - Optimizers: SGD, Adam, RMSProp - Dropout & regularization to prevent overfitting
